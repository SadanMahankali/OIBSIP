# OIBSIP - Data Science Internship

My work for the Oasis Infobyte data science internship.

Track: Data Science. Need to finish at least 3 of the 5 tasks to complete it.

## Tasks
1. Iris Flower Classification (in progress)
2. Unemployment Analysis with Python
3. Car Price Prediction
4. Email Spam Detection
5. Sales Prediction

Each task has its own folder with the notebook, the dataset, and a short README explaining what I did.

## Running it
Everything is Python (Anaconda, 3.13). Install the packages:

    pip install -r requirements.txt

Then open the notebook in whichever task folder you want.
